## ‚ú® Key Features
- **Dynamic Data Fetching:** Seamless communication between the frontend and our custom backend API.
- **Character Directory:** View an extensive list of Harry Potter characters displayed as responsive cards.
- **Detailed Views:** Click on any character card to be redirected to a specific page with in-depth information.
- **Filtering System:** Easily search or filter through the character list to find specific wizards or witches.

## üöÄ Getting Started

### Prerequisites
Make sure you have the following installed:
* [Node.js](https://nodejs.org/) (which includes **npm**)
### Installation & Setup
Follow these steps to get the project running locally:
#### 1. Backend Setup
Navigate to the backend directory and start the development server:
cd Backend
npm install
npm run dev
2. Frontend Setup
Open a new terminal, navigate to the frontend directory, and start the development environment:
cd Frontend
npm install
# To compile TailwindCSS and launch the live server:
npm run dev
Note: If you need to watch for TypeScript changes, run:
npx tsc --watch
